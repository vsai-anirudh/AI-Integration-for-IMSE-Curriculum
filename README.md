# AI Integration for IMSE Curriculum

AI tools and templates for the IMSE curriculum, including NotebookLM setup workflows, Lean Six Sigma materials, an AI-driven resume assistant, and sample project data.

## Repository Structure

- `docs/` – Project documentation (NotebookLM setup, Lean Six Sigma course context, resume assistant overview).
- `templates/` – Lean Six Sigma templates, forecasting template, and resume assistant prompts/guides.
- `data/` – Source course notes and sample datasets used with NotebookLM and templates.

## NotebookLM Materials

- Setup instructions: `docs/notebooklm_setup.md`
- Lean Six Sigma course context: `docs/lean_six_sigma_course_context.md`
- TE533 course note PDFs: `data/TE533_LSS_Notes_Part1.pdf` … `data/TE533_LSS_Notes_Part10.pdf`

## Resume Assistant Materials

- Concept overview: `docs/resume_assistant_overview.md`
- AI prompt for ATS-style analysis: `templates/resume_assistant_prompt.md`
- Resume template and usage guide: `templates/resume_template_guide.md`
